# FASTWAY CARGO – Corporate Logistics Website

A modern, responsive, and conversion-focused corporate website for **FASTWAY CARGO**, a premier global logistics, freight forwarding, and supply chain solutions provider. 

## 🚀 Project Overview
This project is a professional web platform designed to streamline global shipping requests and showcase comprehensive cargo services. The website is engineered to look trustworthy, efficient, and highly reliable for businesses worldwide.

## 🛠️ Core Services Featured
* **Customs Clearance:** Fast-track regulatory compliance, import/export documentation, and tariff consulting.
* **Sea Freight (FCL & LCL):** Full Container Load (FCL) for large shipments and Less than Container Load (LCL) for cost-effective consolidated cargo.
* **Air Freight:** Express global shipping solutions for time-sensitive cargo.
* **Courier Services:** Secure door-to-door domestic and international parcel delivery.

## 🔑 Key Features
* **Interactive Quote Calculator:** A multi-step form capturing origin, destination, dimensions, and service types.
* **Tracking Widget UI:** A placeholder interface for real-time container and parcel tracking.
* **Scannable Grid Layout:** Icon-driven design highlighting key logistics sectors.
* **Fully Responsive:** Optimized for desktop and mobile logistics managers on the go.

## 🎨 Design & Branding
* **Primary Palette:** Professional Navy Blue `#0A192F` (Trust & Authority)
* **Accent Palette:** Safety Orange `#FF6B35` (Speed & Action)
* **Typography:** Clean, geometric sans-serif fonts for maximum readability.

## 📈 Future Enhancements
- Integration of a live tracking API (e.g., AfterShip or ocean carrier webhooks).
- Live freight rate API integration for instant automated quoting.
- Client dashboard to manage past shipments and invoices.
